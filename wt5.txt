AIM:

To display the Image Slide Show Digital clock and Survey Poll to make webpage
dynamic.

PROCEDURE:

• Open VS code editor. Create new file and select the language as “HTML”.
• Create the structure of HTML and include the Style in head as embedded style
sheet that has .fade, @-webkit-keyframe, @keyframes,@media to create slide
show effect.
• In script,using date library inside showTime()function to read local time in
hours,minutes,seconds.if hours>12 then its “PM”; if hours == 0 then its
“AM”.Create 2 functions showTime() and showSlide() for date and to slide
show the image for every second in script.
• Using DOM object document.getElementByclassName using style.display
attribute values,block and active slide show is created. To make the page
dynamic use 2 functions setInterval() and setTimeOut().
• Open a HTML document that includes above style sheets and script using
embedded<script> and <style>tag.
• Save the document as .HTML extensions.
• Run the file by opening the HTML document in browser.

        var slideIndex = 0;

        function showSlides() {
            var i;
            var slides = document.getElementsByClassName("mySlides");
            var dots = document.getElementsByClassName("dot");
            for (i = 0; i < slides.length; i++) {
                slides[i].style.display = "none";
            }
            slideIndex++;
            if (slideIndex > slides.length) { slideIndex = 1 }
            for (i = 0; i < dots.length; i++) {
                dots[i].className = dots[i].className.replace(" active", "");
            }
            slides[slideIndex - 1].style.display = "block";
            dots[slideIndex - 1].className += " active";
            setTimeout(showSlides, 1000); // Change image every 1 seconds
        }
